# CommonHE v1.0.1 发版记录

发布日期：2026-05-22

## 发版范围

- 版本号从 `1.0.0` 升级到 `1.0.1`。
- 将桌面端 LLM 响应处理修复同步到公开仓库。
- 本次发版不改变产品定位：仍是 `星星的vibecoding启动器` 的公开工程包，用于生成 Codex 接手协作包。

## 本次修复

- 增强 `梦星星` 主 Agent 响应解析与结构化修复流程。
- 只在最终三方案阶段执行严格 `solutions` JSON 合同修复，避免影响普通澄清对话。
- 增加 DeepSeek v4-pro / v4-flash 在空响应、SSE 响应、Markdown 三方案输出等场景下的回归保护。
- 增加 MiniMax custom provider 的 `<think>...</think>` 包裹响应测试覆盖，并避免影响 GPT / DeepSeek / Codex 的原有解析路径。
- 增加模型调用日志，运行时写入启动器自身 `data/logs/commonhe-agent-calls.jsonl`，不写入目标项目目录。
- 前端错误提示增加脱敏诊断摘要，方便定位 `status`、`endpoint`、`reason` 等失败原因。
- `.gitignore` 增加本地验证/构建产物忽略规则：`tmp/`、`apps/desktop/src-tauri/target/`、`apps/desktop/src-tauri/gen/`、`apps/desktop/src-tauri/resources/commonhe/`、`release/*/`。
