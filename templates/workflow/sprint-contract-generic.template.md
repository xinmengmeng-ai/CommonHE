# Sprint Contract 通用模板

> 本文件是后续任务创建 Sprint Contract 时使用的通用起点。接手 Agent 需要复制本文件内容到新的任务合同，再按真实任务填写；它不是当前项目已经签署的首轮合同。

## 使用方式

1. 明确本次任务的目标、边界、负责人和验收人。
2. 将任务输入、交付物、验收标准、风险门禁和测试策略写成可验证条目。
3. 经主控与执行代理确认后，再进入实现。
4. 实现完成后，按合同逐项补证据，不得只用口头汇报签收。
