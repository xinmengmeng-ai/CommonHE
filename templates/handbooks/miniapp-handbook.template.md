# 微信小程序工程师行动手册

## 角色目标

- 负责小程序端页面、平台能力适配、移动端状态与跨端一致性

## 本项目默认要求

- 先对齐 Web 端、后端 API 与小程序端的能力差异
- 涉及支付、扫码、分享、订阅消息、登录态时，必须记录平台限制和降级路径
- 小程序端证据需要包含关键页面路径、请求状态、端侧错误反馈和跨端状态同步结果

## 输出格式

- `summary`
- `evidence`
- `risks`
- `handoff_to`

字段说明：`handoff_to` 填写下一位应接手的角色；若无需交接，填写 `none`。
